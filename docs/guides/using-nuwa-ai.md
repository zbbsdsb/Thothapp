# Using Thoth's AI

Thoth's AI analysis is powered by Google Gemini 3.1 Flash (and optionally your own Minimax API key). Let's explore how it works and how to get the best results!

## How AI Analysis Works

Every dream you archive is automatically analyzed by Thoth's AI, which produces three key outputs:

1. **Imagery Tags**: 3-5 single-word tags that capture the key symbols and themes in your dream
2. **Psychological Insight**: A short, poetic insight about what your dream might mean
3. **Divine Oracle**: A mystical, cryptic sentence inspired by Tarot and ancient scripts

## Public Quota vs. Private API Key

Thoth offers two ways to use AI analysis:

### Public Quota (Default)
- Uses Thoth's shared Gemini API key
- Limited to 3 dreams per day (check your remaining quota in the footer or Settings tab)
- Perfect for trying out Thoth

### Private Minimax API Key
- Add your own API key in the **Settings** tab
- Removes daily quota limits
- All analysis uses your own key for full privacy

To add your key:
1. Go to the **Settings** tab
2. Find the "AI Synthesis" section
3. Enter your Minimax API key in the password field
4. The key is automatically saved to your profile

## Tips for Best Results

To get the most insightful AI analysis:

1. **Be Detailed**: Include as many specific details as possible when describing your dream — places, people, emotions, colors, and sensations all help the AI understand your dream better
2. **Speak Clearly**: If using voice entry, speak clearly and at a natural pace
3. **Be Honest**: The AI works best with genuine, unfiltered dream descriptions
4. **Review Patterns**: Look at multiple dreams together to spot recurring themes in your tags and insights

## Privacy Note

Your dreams and AI analysis are stored securely in Firestore. If you use your own API key, all analysis is processed using your key directly.
