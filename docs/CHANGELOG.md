# Changelog

All notable changes to the Thoth project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [2.0.0] - 2026-04-23

### Added
- Monorepo architecture using Nx
- CI/CD pipeline with GitHub Actions
- Core services package structure
- Shared UI components package
- Shared TypeScript types package
- Shared configuration package
- Documentation structure
- Unit test setup with Jest
- ESLint configuration
- Comprehensive technical documentation

### Changed
- Updated project structure to support multiple platforms
- Enhanced API documentation
- Improved README with architecture information
- Updated technical documentation with CI/CD details
- Updated version numbering

### Fixed
- Various bug fixes and performance improvements

## [1.0.8] - 2026-04-19

### Added
- Initial release of Thoth AI Dream Archive
- Core dream capture and analysis functionality
- AI-powered transcription and analysis
- Divine Oracle feature
- Memory Collapse Countdown
- Global Imagery Map
- User authentication with Google
- Firebase Firestore integration
- Cloudflare R2 storage integration
- Web application interface

### Changed
- Initial implementation

### Fixed
- Initial release
