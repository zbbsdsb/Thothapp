# dreambase: The Imagery Infrastructure

## Overview
**dreambase** is the underlying database and imagery infrastructure for the **Thoth** ecosystem. While Thoth provides the user interface, AI analysis, and recording tools, dreambase serves as the repository for "atomic dream information."

## Atomic Imagery Concept
**dreambase** focuses on "Atomic Imagery"—the smallest, most fundamental symbols and themes that make up our dreams. By extracting these atoms (e.g., "water," "clock," "flight," "shadow"), we can build a structured map of the human subconscious.

## Relationship
- **Thoth**: The application layer. It handles user authentication, voice recording, AI transcription, and the "Imagery Hall" visualization.
- **dreambase**: The data layer. It stores the extracted symbols, keywords, and transcripts as structured, atomic information.

## Vision
We believe that the imagery stored in dreambase will become a core infrastructure for the future metaverse. Imagine a virtual world where the environments and objects are generated directly from the collective subconscious of its users.

**Important Note**: dreambase is a component of Thoth. It is not a standalone application, but rather the engine that powers Thoth's archiving capabilities.
