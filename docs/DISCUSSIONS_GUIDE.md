# GitHub Discussions 使用指南

欢迎来到 Thoth 社区！本指南将帮助你了解如何有效地使用 GitHub Discussions 与社区互动。

## 📋 Discussions 分类

我们将 Discussions 分为以下几个类别：

### 1. **General** 💬
- 与 Thoth 相关的一般讨论
- 分享你的使用体验
- 社区闲聊和社交互动

### 2. **Ideas** 💡
- 提出新功能想法
- 讨论现有功能的改进
- 在正式提 Issue 前先在这里讨论想法

### 3. **Q&A** ❓
- 寻求使用 Thoth 的帮助
- 回答其他社区成员的问题
- 技术问题解答

### 4. **Show and tell** 🎉
- 分享你使用 Thoth 创建的内容
- 展示你的项目
- 分享技巧和最佳实践

### 5. **Announcements** 📢
- 官方公告和更新
- 发布信息
- 重要社区通知（仅维护者可发布）

## 🚀 如何开始

1. **浏览现有的讨论**：查看是否已经有相关的讨论
2. **创建新讨论**：点击 "New discussion" 按钮
3. **选择合适的分类**：确保你的讨论放在正确的分类下
4. **清晰描述**：提供详细的描述，帮助其他人理解你的问题或想法

## 🤝 社区行为准则

- 保持友善和尊重
- 使用包容性的语言
- 接受不同的观点
- 专注于对社区最有利的事情
- 对其他社区成员表示同理心

## 📌 小贴士

- **搜索**：在创建新讨论前，先搜索一下是否已经有类似的讨论
- **标签**：使用相关的标签来帮助分类你的讨论
- **回复**：如果你看到可以帮助的讨论，请参与进来！
- **投票**：对你喜欢的想法或答案进行投票

## 🔗 其他资源

- [Discord 服务器](https://discord.gg/[邀请码])：实时社区交流
- [问题报告](https://github.com/[用户名]/Thothapp/issues/new/choose)：提交 Bug 或 Feature Request
- [贡献指南](CONTRIBUTING.md)：了解如何为项目做贡献

让我们一起打造一个热情友好的社区！🎉
