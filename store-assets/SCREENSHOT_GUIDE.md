# 📸 截图指南 - Thoth Google Play 上架

## 截图规格要求
- **数量**: 最少 2 张，最多 8 张（手机）
- **尺寸**: 9:16 竖屏（推荐 1080×1920）或 16:9 横屏（1920×1080）

- **内容**: 必须是真实应用截图，不能使用人工渲染图

## 📱 截图步骤

### 方法 A：Android 模拟器（推荐）
1. 启动 Android Studio
2. 打开 AVD Manager（Tools > Device Manager）
3. 启动一个 Pixel 系列模拟器（推荐 Pixel 8 Pro）
4. 安装并运行 Thoth APK/AAB
5. 使用模拟器的截图按钮（右侧工具栏）截图

### 方法 B：物理设备
1. 启用开发者选项（设置 > 关于手机 > 连点版本号 7 次）
2. 启用 USB 调试
3. 连接电脑，运行 `adb devices` 确认连接
4. 安装 Thoth：`adb install app-debug.apk`
5. 使用音量减 + 电源键截图，或从通知栏分享到电脑

### 方法 C：使用 Android Studio 的 Device Explorer
1. 运行应用
2. View > Tool Windows > Logcat
3. 使用截图按钮（屏幕图标）

## 🖼️ 截图后处理

1. **重命名**: 使用上述推荐文件名

3. **压缩**: 使用 TinyPNG 或类似工具压缩（可选）
4. **放置目录**: 复制到 `store-assets/screenshots/` 目录
