# Thoth Deployment Plan

This document outlines the steps required to deploy and maintain the **Thoth Subconscious Archive System**.

## 1. Environment Configuration

### Gemini API (Hybrid Mode)
- **Developer Key**: `GEMINI_API_KEY` (Set in AI Studio Settings). This provides a baseline quota for all users.
- **User Key**: `API_KEY` (Injected when a user selects their own key via the UI).
- **Logic**: The app prioritizes the user's personal key if available, falling back to the developer's key. This allows "power users" to use their own quota while keeping the app accessible to everyone.
- **Action**: Ensure `GEMINI_API_KEY` is set in your Settings menu.

### Firebase Configuration
- **File**: `firebase-applet-config.json`
- **Status**: Automatically generated by the `set_up_firebase` tool. 
- **Verification**: Ensure `apiKey`, `authDomain`, `projectId`, `appId`, and `firestoreDatabaseId` are present and valid.

## 2. Firebase Services Setup

### Authentication
- **Provider**: Google Sign-in.
- **Action**: Go to the Firebase Console -> Authentication -> Sign-in method and ensure Google is enabled.

### Firestore Database
- **Rules**: Deploy the `firestore.rules` file using the `deploy_firebase` tool.
- **Structure**: Defined in `firebase-blueprint.json`.
  - `dreams` collection: Stores transcripts, audio URLs, and AI tags.
  - `users` collection: Stores user profiles (if applicable).

### Cloud Storage
- **Path**: `dreams/{userId}/{dreamId}.{ext}`
- **Rules**: Ensure storage rules allow authenticated users to read/write to their own folder.
  - *Recommended Rules*:
    ```
    service firebase.storage {
      match /b/{bucket}/o {
        match /dreams/{userId}/{allPaths=**} {
          allow read, write: if request.auth != null && request.auth.uid == userId;
        }
      }
    }
    ```

## 3. Build & Verification

### Local Verification
- Run `npm run lint` to ensure code quality.
- Run `npm run build` to verify the production build process.

### Deployment
- The application is deployed to Cloud Run via the AI Studio "Share" or "Deploy" workflow.
- Ensure the `dist/` directory is correctly generated as the application is a React SPA.

## 4. Maintenance & Quotas

- **Firestore**: Monitor usage in the Firebase Console. Thoth uses the Enterprise (Spark) tier by default.
- **Gemini API**: Be aware of rate limits for the `gemini-3-flash-preview` model.
- **Storage**: Audio files are stored as `.webm`/`.ogg`. Monitor storage bucket size if usage scales.
